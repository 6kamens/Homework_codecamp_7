grid และ flex แตกต่างกันอย่างไร grid จะสามารถจัดการ layout ได้ในเเบบ 2 มิติ คือกำหนดพื้นที่ได้ทั้งเเกน x เเละ y ในขณะที่ flex จะทำการกำหนดได้เเค่จะเป็นเเนวนอนหรือเเนวตั้งเท่านั้น
justify and align แตกต่างกันอย่างไร justify เป็นการจัดการตำเเหน่งในเเนวนอน (เเกน x) ส่วน align เป็นการจัดการตำเเหน่งในเเนวตั้ง (เเกน y)
position คืออะไร และ อะไรบ้าง position คือการกำหนดประเภทการวางตำเเหน่งของ element
  • static : ค่าตำเเหน่ง default
  • relative : การวางตำเเหน่งให้เทียบกับหน้าหลัก
  • fixed : เป็นการตรึง element นั้นให้อยู่ที่เดิมของหน้าจอตลอดไม่ว่าจะเลื่อนไปไหน
  • absolute : การวางตำเเหน่งเทียบกับ relative เป็นหลักในการกำหนดตำเเหน่ง
  • sticky : เป็นการวางตำเเหน่งที่จะยังติดไปเเม้มีการเลื่อนหน้า
z-index คืออะไรใช้อย่างไร เป็นจำเเหน่งในการมองมุมลึกเข้าไป หรือระดับชั้น layer ไว้ให้ element ไหนอยู่ทับ element ไหน
