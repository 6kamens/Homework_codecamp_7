CSS syntax ประกอบด้วยอะไรบ้าง

selector : เป็นประเภทของ element ที่เลือกใช้ css นี้

property : เป็นประเภทของสิ่งที่อยากให้ selector ที่เลือกมานั้นเปลี่ยน เช่น color , font-size เป็นต้น

value : ค่าของ property นั้นๆเช่น color ก็จะมี value เป็นสีต่างๆ
การเขียน CSS มีกี่แบบอะไรบ้าง

3 เเบบ

  1. inline css : เขียนข้างใน tag เลย
  2. internal css : เขียน style บน head
  3. external css : เขียน file .css เเละค่อยเรียกใช้
selector มีกี่แบบอะไรบ้าง

3 เเบบ

  1. class selectors : ชื่อ class ที่ตั้งให้กับ tag
  2. id selectors : id ที่ตั้งให้กับ tag
  3. tag selectors : tag html
combinator มีกี่แบบอะไรบ้าง

4 เเบบ

  1. descendant selector (space) : ทำการเลือก element ภายใน element นั้นอีกที ทั้งหมด ไม่ว่า elemetn ภายในจะอยู่ชั้นลึกเเค่ไหน
  2. child selector (>) : ทำการเลือก element ภายใน element นั้นอีกที เฉพาะชั้นนอกที่ติดกับ element หลัก
  3. adjacent sibling selector (+) : ทำการเลือก element ที่เป็น combination ตามนั้น
  4. general sibling selector (~) : ทำการเลือก element ทั้งหมดที่ต่อจาก element หลักที่เลือก
